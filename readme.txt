== Description ==

Gravity Flow is an Add-On for [Gravity Forms](http://www.stevenhenty.com/gravityforms)

Facebook: [Steven Henty](https://www.facebook.com/hentydevelopment)

= Requirements =

1. [Purchase and install Gravity Forms](http://www.stevenhenty.com/gravityforms)
2. Wordpress 4.2+
3. Gravity Forms 1.9.4+


= Support =
If you find any that needs fixing, or if you have any ideas for improvements, please get in touch:
http://www.stevenhenty.com/contact/


== Installation ==

1.  Download the zipped file.
1.  Extract and upload the contents of the folder to /wp-contents/plugins/ folder
1.  Go to the Plugin management page of WordPress admin section and enable the 'Gravity Forms Approval Add-On' plugin

== Frequently Asked Questions ==

= Which license of Gravity Forms do I need? =
Gravity Flow will work with any license of [Gravity Forms](http://www.stevenhenty.com/gravityforms).
